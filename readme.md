A fluent validation framework for javascript.

A loose port of Jeremy Skinner's Fluent Validation framework to JavaScript.

Uses ES2015 and Babeljs.

## Fluent Validators

isNotNullOrUndefined()

length(min, max)
isNotEmpty()
must( function() {} )
lessThan()
lessThanOrEqual
greaterThan()
greaterThanOrEqual()
equal()
